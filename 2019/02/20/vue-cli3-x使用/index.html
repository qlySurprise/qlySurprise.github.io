vue-cli3.x使用

二月 20, 2019

关于旧版本

Vue CLI 的包名称由 vue-cli 改成了 @vue/cli。 如果你已经全局安装了旧版本的 vue-cli (1.x 或 2.x)，你需要先通过 npm uninstall vue-cli -g 或 yarn global remove vue-cli 卸载它。

执行：

vue –version 控制台打印如下：

3.0.0-rc.10， 说明已安装成功。

二、创建项目

vue create hello-vue-cli3

cd到新建的工程目录：

cd hello-vue-cli3

执行：

npm run serve

在浏览器中打开“http://localhost:8080/”，一个初始化好的vue工程就完成了。

三、使用GUI
通过vue ui命令可以在浏览器中直接打开 vue命令行工具的可视化界面。
